# Measured GPU test results

"Fits 120s" is for the *warm* run at the tested output length; see notes for models whose real-length output would exceed the budget.

Notes:
- VRAM is the resident size reported by Ollama (`/api/ps`), i.e. weights + KV/context.
- `qwen3-vl:8b` used ~9.5 GB, so it needs a 12 GB card, not the 8 GB minimum.
- `qwen3-vl:32b` (dense) ran at only ~3 tok/s on a 24 GB card: a full 4096-token answer would take ~20 min, far past the 120 s budget. Use the MoE `qwen3-vl:30b` on 24 GB, or run the 32B on an 80 GB card.
- Pull time is a one-time per-worker download; it is not part of a job's compute window.
